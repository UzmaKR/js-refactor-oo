Throwing the dice

View project onGitHub

Goal: This is a refactor exercise, taking a DOM-oriented javacript and creating object-oriented javascript from it.

Game: The game is straight-forward. Create a die and roll it. You can create as many die as you want. Rolling the die will roll all the dice and randomly generate numbers on the dice faces.

How to play the game: Load the app.html and play. The supporting JS is application.js.
